What is the CAST Plug-in for Eclipse?

The CAST Plug-in for Eclipse is a software plug-in designed by CAST for use with the Eclipse IDE. This particular release provides the following:

  • visualization of all Java based code violations listed in the CAST Dashboard Service via a web service installed on a J2EE compliant application server (such as Apache Tomcat)
  • bookmarking of violations in code
  • ability to view detailed information about the violation
  • direct link into the CAST Dashboard for further information
