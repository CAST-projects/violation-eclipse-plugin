CAST Plug-in for Eclipse

The CAST Plug-in for Eclipse is in fact several plug-ins that can be added to Eclipse using the Software Updates and Install new Software options within Eclipse. See Installing and configuring the CAST Plug-in for Eclipse for more information.

Web Service

The web service is shipped in the form of a .WAR file (called CASTWS.war) that is delivered as part of the CAST AIP suite or directly from CAST’s Product Center.

CAST Dashboard Service

The CAST Dashboard Service should already be populated with data from your ongoing analyses. It should also contain violations (i.e. code that violates CAST Quality Rules) that have been added to the Action Plan – you can verify this using the CAST Dashboard.

CAST Dashboard

The CAST Dashboard needs to be fully functioning.

If you want (optional) to browse the violations for an object or the objects that violate a specific Quality Rule (through the Eclipse browser) as displayed in the CAST Dashboard, any developer computer that needs to use the CAST Plug-in for Eclipse must also be able to connect to the CAST Dashboard and the CAST Web Service.
