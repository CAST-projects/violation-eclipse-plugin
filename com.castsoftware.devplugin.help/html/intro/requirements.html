Environment Requirements

To use the CAST Plug-in for Eclipse the following requirements are necessary:

CAST

The plug-in is designed to function with the following versions of CAST:

  • 6.4.x
  • 7.0.x

This means you should have a fully functioning CAST installation in one of the supported versions. Please see the relevant CAST Installation Guide for more information about the installation process for this version.

You also need to have successfully worked through the process of populating your CAST Dashboard Service (formerly known as the Central) with information that can be accessed via the CAST Dashboard (i.e. have analyzed your source code, generated a snapshot and added violations to the Action Plan in the CAST Dashboard).

RDBMS

The plug-in functions on the following supported RDBMS (please see the relevant CAST Release Notes to find out which RDBMS version is supported):

CAST version Supported RDBMS
6.4.x
  • Oracle Server
  • Microsoft SQL Server
7.0.x
  • All supported RDBMS including the CAST Storage Service.
Web Service

The CAST Plug-in for Eclipse interacts with the CAST Dashboard Service using a web service installed on a J2EE compliant application server (such as Apache Tomcat). As such, the web service must already be installed and configured.

  • For Oracle Server environments, when configuring the web service, you need to specify the schema, user ID and password that correspond to the schema in which the CAST Dashboard Service has been installed.
  • For Oracle Server environments, you must make sure that the user ID you specify when configuring the web service has the following rights/roles:
    • Oracle 10g2 = ALTER SESSION. Note that this role is implicitly granted in Oracle 10.2.0.1
    • Other Oracle versions = CONNECT

    This is because the CAST Plug-in for Eclipse modifies the Oracle Optimizer Mode using the following query:

    alter session set optimizer_mode=ALL_ROWS
J2EE compliant application server

The CAST Plug-in for Eclipse interacts heavily with the CAST Dashboard and a web service - both of which are installed on a J2EE compliant application server (such as Apache Tomcat). Please see the relevant CAST Release Notes to find out which J2EE compliant application version is supported for CAST 6.4.x and 7.0.x.

Eclipse

The plug-in is supported on:

  • Eclipse 3.4
  • Eclipse 3.5
  • Eclipse 3.6

You can download Eclipse for free: http://www.eclipse.org/downloads/ CAST recommends the Eclipse IDE for Java EE Developers, although the CAST plug-in will work with other packages.
