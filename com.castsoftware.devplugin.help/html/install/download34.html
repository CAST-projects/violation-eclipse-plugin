Plug-in installation for Eclipse 3.4.x

To install the CAST Plug-in for Eclipse in Eclipse 3.4.x:

  • In Eclipse, please open the Software Update and Add-ons manager. This can be accessed via the Help > Software Updates menu option. This will launch a new dialog box:
  • Click the Available Software tab:
  • Then click Add Site. A new dialog box will then be displayed:

  • Click the Archive button to add the file location of the ZIP file containing the plug-ins.
  • Click OK.
  • The site will then be added to the list of existing sites (if any) and any available components will be listed:

  • Select all components (as shown below). Make sure other unrelated components are not selected. Then click Install (top right hand corner) to start the installation process:

  • At this point Eclipse will verify the contents of the ZIP and display a new dialog box with a list of all available software from CAST. Review the items that will be installed and then click Finish:

  • The installation process will then begin:

  • When the installation process is complete, Eclipse will ask you whether you want to restart the program. CAST recommends that you do so – click Yes in the displayed dialog box:

Please refer to Plug-in configuration for more information about post install configuration process.
