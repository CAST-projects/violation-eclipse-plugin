How to manually fetch the violations

Although the CAST Violation List will automatically update when you make a selection in the Fetching Parameters dialog box, you can also force Eclipse to fetch violations from the CAST Dashboard Service at any time. This can be particularly useful if the contents of the CAST Dashboard Service have been updated following a new analysis.

To do so:

  • Click the button located in the CAST Violation List (note that this button is greyed until you select at least one Application/Module and one Technology in the Fetching Parameters dialog box).

Notes

  • Please remember that only Quality Rule based violations will be fetched into the CAST Violation List and not metric based violations.
